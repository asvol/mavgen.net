# mavgen.net
Universal MAVLink message generator for .NET\Mono

Use [liquid](http://dotliquidmarkup.org/) syntax for message generation.

Example usage:

```bash
mavgen-net.exe gen -t=standard.xml -i=in -o=out -e=cs csharp.tpl
```

or see example

```bash
mavgen-net.exe example
```
